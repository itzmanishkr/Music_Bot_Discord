# Music_Bot_Discord
A feature-rich Discord music bot built with Python, supporting queue, skip, pause/resume and YouTube search.

# Installation

> Download ffmpeg > Extract and copy to the C drive > Add bin path to system variables
> Install all the libraries in the requirements.txt
> Change the name of .env.example to .env and set the token generated on your discord dev account
> Run the program.
